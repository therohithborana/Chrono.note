# **App Name**: ChronoNote

## Core Features:

- Timestamped Notes: Automatically timestamp each note entry as it's created, including the duration between each timestamped note.
- Text Export: Allow users to export their notes as a plain text file for sharing or archiving.
- Share via URL: Generate a unique, public URL for sharing notes online without any external database.
- Hide Notes Hotkey: Implement a hotkey to blur the notes for privacy.

## Style Guidelines:

- Primary color: Calm Blue (#64B5F6) to provide a serene and focused note-taking environment.
- Background color: Light Gray (#F5F5F5) for a clean and distraction-free interface.
- Accent color: Soft Green (#A5D6A7) to highlight important tags or sections.
- Body and headline font: 'PT Sans', a sans-serif for a modern yet approachable look.
- Minimalist icons for filtering, exporting, and sharing functionalities.
- Clean and simple layout, emphasizing a chronological flow of notes with clear timestamp labels.
